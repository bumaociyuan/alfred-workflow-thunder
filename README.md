## Introduction
在Alfred搜索框中输入`xz`(即下载首字母)，然后回车，自动打开迅雷并打开添加下载对话框，复制粘贴板中的网址，并将焦点定位到`立即下载`按钮。
然后回车确认即开始下载。

`opt+shift+x` 榨干网解析网盘

## Requirement
本workflow使用的Applescript脚本需要系统辅助权限，请到`偏好设置 - 安全与隐私 - 辅助`添加Alfred程序。

```
pip install requests
```
